React Introduction

What is React?

  1. React is a JavaScript library for building user Interfaces.
  2. A Open Source framework maintained by facebook and a community under MIT license.
  3. React was created by Jordan Walke, as software engineer at facebook.

Advantages of React

  1. Build reusable UI components.
  2. React provides a declarative API which makes writing application a lot easier.
  3. We can build high performance and interactive web applications.
  4. Build Single Page Application (SPA).
  5. It uses virtualDOM for DOM manipulation rather than realDOM.
  6. Support Server side rendering (SSR) which improves SEO.
  7. We can design complex react application using additional libraries for state management (redux library) and routing(react-router library).
